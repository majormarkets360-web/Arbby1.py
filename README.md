# AI Crypto Arbitrage Scanner

Real-time cryptocurrency arbitrage scanner with AI-powered predictions.

## Deployment

This app is deployed on Streamlit Cloud. Visit [share.streamlit.io](https://share.streamlit.io) to deploy your own version.

## Local Development

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
